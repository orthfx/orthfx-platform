# Orthodox Registry

A directory and interactive mapping system for Orthodox Christian communities, parishes, and monasteries.

Part of the **orthfx-platform** monorepo. It uses data from the shared `@orthfx/backend` and components from `@orthfx/ui`.

## Features

- Interactive Map powered by Leaflet
- Ingestion scripts for scraping and processing community data
- Search and filtering for parishes by jurisdiction and location

## Development

```bash
# Start the Vite+ dev server
vp dev
```

## Deployment

Deploys automatically to Netlify on push to `main`.

- **Site:** orthfx-platform-registry
- **URL:** https://orthfx-platform-registry.netlify.app
- **Env vars:** none required
