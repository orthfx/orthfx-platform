# orthfx.com

The official website for orthfx, an organization dedicated to building Orthodox Christian digital tools and resources.

This frontend serves as the landing page for the organization, detailing our mission, open-source projects, and contact information.

## Development

```bash
# Run the Vite+ dev server
vp dev
```

## Deployment

Deploys automatically to Netlify on push to `main`.

- **Site:** orthfx-platform-website
- **URL:** https://orthfx-platform-website.netlify.app
- **Env vars:** none required
